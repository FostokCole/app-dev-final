# Hospital Management System
A Windows Forms Application built to provide Hospitals with a simple way to manage patients, doctors and appointments with CRUD functions.

## Features
- Add, update and delete patients
- Manage various types of records
- Schedule appointments
- Search functionality using LINQ
- Persistent database storage

## Project Structure
- Forms/ -> All UI/Form classes
- Models/ -> Data classes (Patient, Doctor, Appointment, MedicalRecord)
- HospitalService -> Buisness Logic class
- Program.cs -> Entry point for application
